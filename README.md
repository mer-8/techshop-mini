# TechShop Mini

A professional, responsive e-commerce web application built to practice modern web development concepts. This project implements a shopping cart with functional state management and communicates with a custom API route.

## 🚀 Key Features

*   **Catalog Management**: Browse products dynamically.
*   **Shopping Cart**: Add, update, and remove items with real-time total calculation.
*   **State Management**: Advanced use of React `useState` and `useEffect` hooks.
*   **API Integration**: Communication between a custom Next.js API route and the frontend.
*   **Responsive UI**: Modern, clean design using Tailwind CSS.

## 🛠 Tech Stack

*   **Framework**: [Next.js](https://nextjs.org/) (App Router)
*   **Language**: [TypeScript](https://www.typescriptlang.org/)
*   **Styling**: [Tailwind CSS](https://tailwindcss.com/)
*   **Deployment**: [Vercel](https://vercel.com/)
